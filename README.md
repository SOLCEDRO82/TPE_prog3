## Trabajo Práctico Especial - Programación 3 (2026)
### TUDAI - UNICEN
Este repositorio contiene la resolución del Trabajo Práctico Especial de la cursada 2026. El objetivo del proyecto es gestionar la carga, consulta y asignación eficiente de paquetes en camiones de reparto, aplicando diversas técnicas algorítmicas y realizando una correcta toma de decisiones sobre las estructuras de datos a utilizar en cada caso para garantizar la eficiencia.
